# CATOCA — Система ТОиР

Приложение для планирования и управления техническим обслуживанием и ремонтом оборудования горно-рудного операции «Катока».

## Технологии

- **React 19** + **Vite 8**
- **Tailwind CSS v4** (темная индустриальная тема)
- **React Router v7** (клиентский роутинг)
- **date-fns** (работа с датами)
- **lucide-react** (иконки)

## Модули

| Маршрут | Описание |
|---------|----------|
| `/` | Главный дашборд (статистика, SVG-диаграмма) |
| `/equipment` | База оборудования (иерархия до 6 уровней) |
| `/tickets` | Тикеты аварий и ППР |
| `/materials` | Заявки на ТМЦ (снабжение) |
| `/reports` | Отчёты по простоям (неделя/месяц/год) |
| `/knowledge` | База знаний (инструкции, схемы) |

## Роли пользователей

| Логин | Пароль | Роль |
|-------|--------|------|
| `admin` | `1111` | Администратор (полный доступ) |
| `chefe` | `2222` | Руководитель |
| `dispacho` | `3333` | Диспетчер (управление статусами) |
| `user` | `4444` | Инженер (создание тикетов) |
| `supply` | `5555` | Снабжение |

## Запуск

```bash
npm install
npm run dev
```
